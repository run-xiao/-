# 灵动岛监控系统 - Dynamic Island Monitor

一个仿macOS灵动岛的Windows系统监控工具，采用现代化UI设计。

## 功能特性

- 🎨 **圆角窗口设计** - 美观的圆角透明窗口
- 🌓 **主题切换** - 深色/浅色模式自由切换
- 🔍 **透明度调节** - 50%-100%可调透明度
- ⚡ **刷新频率** - 可自定义刷新间隔（0.5s-3s）
- 📊 **灵活显示** - 可选择显示CPU、内存、电池信息
- ⏰ **闹钟提醒** - 设置闹钟，到时间自动提醒
- 📝 **备忘录** - 记录重要事项，随时查看
- 🔔 **系统托盘** - 最小化到托盘，后台运行

## 安装依赖

```bash
pip install -r requirements.txt
```

或者手动安装：

```bash
pip install psutil pystray Pillow
```

## 运行程序

```bash
python LDD.py
```

## 使用说明

### 基本操作

- **右键点击** - 展开/收起详细信息
- **拖动窗口** - 移动灵动岛位置（松开后自动居中）
- **系统托盘** - 右键托盘图标可打开设置或退出

### 设置功能

#### 外观设置
- 切换深色/浅色主题
- 实时预览主题效果

#### 透明度设置
- 滑块调节窗口透明度（50%-100%）
- 实时预览透明度变化

#### 刷新频率
- 选择数据刷新间隔
- 较短间隔更实时，较长间隔更省资源

#### 显示内容
- 勾选需要显示的信息项
- CPU使用率
- 内存使用率
- 电池状态

#### 闹钟提醒
- 添加闹钟：设置时间和标签
- 删除闹钟：选中后点击删除
- 到时间自动弹出提醒窗口

#### 备忘录
- 添加备忘：输入标题和内容
- 删除备忘：选中后点击删除
- 列表显示所有备忘事项

## 技术架构

### 核心组件

- **GUI框架** - Tkinter（Python内置）
- **系统托盘** - pystray库
- **系统监控** - psutil库
- **Windows API** - ctypes调用

### 文件结构

```
编程/
├── LDD.py              # 主程序文件
├── settings.json       # 配置文件（自动生成）
├── requirements.txt    # Python依赖
└── README.md          # 说明文档
```

## 配置说明

程序会自动生成`settings.json`配置文件，包含以下设置：

```json
{
  "is_dark": true,
  "opacity": 0.95,
  "refresh_interval": 1000,
  "show_cpu": true,
  "show_ram": true,
  "show_battery": true,
  "alarms": [],
  "memos": []
}
```

## 注意事项

1. **纯Tkinter实现** - 无需额外安装GUI框架，开箱即用
2. **Windows专用** - 目前仅支持Windows系统
3. **管理员权限** - 某些功能可能需要管理员权限
4. **防火墙** - 首次运行可能需要允许通过防火墙

## 常见问题

### Q: 为什么任务栏没有显示？
A: 这是故意设计的，灵动岛作为悬浮窗不应占用任务栏空间。

### Q: 如何完全退出程序？
A: 右键系统托盘图标，选择"退出"。

### Q: 设置保存到哪里了？
A: 保存在程序同目录下的`settings.json`文件中。

### Q: 可以自定义窗口大小吗？
A: 目前窗口大小固定，后续版本可能添加自定义选项。
