# 管理器窗口模块化重构说明

## 📁 文件结构

本次重构将原 `manager_windows.py` 中的三大管理类拆分为独立文件，提高了代码的可维护性和模块化程度。

### 新文件结构

```
文件/
├── LDD.py                          # 主程序文件（灵动岛）
├── manager_windows.py              # 闹钟服务类 + 统一导入接口
├── manager_utils.py                # 通用工具函数模块
├── alarm_manager_window.py         # 闹钟管理窗口类
├── memo_manager_window.py          # 备忘录管理窗口类
├── message_center_window.py        # 消息中心窗口类
├── settings_window.py              # 设置窗口（保持不变）
└── data/
    ├── alarms.json                 # 闹钟数据
    ├── memos.json                  # 备忘录数据
    └── messages.json               # 消息数据
```

## 📦 模块说明

### 1. `manager_utils.py` - 通用工具模块
**功能：** 提供所有管理器共享的工具函数

**包含内容：**
- 数据加载/保存函数：`load_alarms()`, `save_alarms()`, `load_memos()`, `save_memos()`, `load_messages()`, `save_messages()`
- 消息管理：`add_message()`
- 音频播放：`play_custom_audio()`, `stop_all_sounds()`
- 音频文件扫描：`scan_sound_files()`
- 常量定义：数据文件路径、音频文件夹路径、支持的音频格式

### 2. `alarm_manager_window.py` - 闹钟管理窗口
**功能：** 独立的闹钟管理界面

**主要类：**
- `AlarmManagerWindow`: 提供闹钟的增删改查、启用/禁用、声音配置等功能

**依赖：**
- `manager_utils`: 数据加载/保存、音频播放

### 3. `memo_manager_window.py` - 备忘录管理窗口
**功能：** 独立的备忘录管理界面

**主要类：**
- `MemoManagerWindow`: 提供备忘录的增删改查功能

**依赖：**
- `manager_utils`: 数据加载/保存

### 4. `message_center_window.py` - 消息中心窗口
**功能：** 显示系统通知和历史记录

**主要类：**
- `MessageCenterWindow`: 提供消息查看、标记已读、删除等功能

**依赖：**
- `manager_utils`: 数据加载/保存

### 5. `manager_windows.py` - 入口模块（保留）
**功能：** 提供 AlarmService 类和统一的导入接口

**包含内容：**
- `AlarmService` 类：闹钟检查、触发、通知显示的核心业务逻辑
- 统一导入语句：方便其他模块一次性导入所有需要的类和函数

**为什么保留：**
- `AlarmService` 是核心业务逻辑，与 UI 解耦
- 作为模块入口点，提供清晰的导入路径

## 🔄 导入关系

```
LDD.py (主程序)
  ├─> manager_windows.AlarmService (闹钟服务)
  ├─> alarm_manager_window.AlarmManagerWindow (闹钟管理UI)
  ├─> memo_manager_window.MemoManagerWindow (备忘录管理UI)
  ├─> message_center_window.MessageCenterWindow (消息中心UI)
  └─> manager_utils (数据加载函数)

alarm_manager_window.py
  └─> manager_utils (数据操作、音频播放)

memo_manager_window.py
  └─> manager_utils (数据操作)

message_center_window.py
  └─> manager_utils (数据操作)

manager_windows.py
  ├─> manager_utils (工具函数)
  ├─> alarm_manager_window.AlarmManagerWindow
  ├─> memo_manager_window.MemoManagerWindow
  └─> message_center_window.MessageCenterWindow
```

## ✅ 优势

### 1. **模块化设计**
- 每个窗口类独立成文件，职责清晰
- 便于单独维护和测试

### 2. **代码复用**
- 通用工具函数集中在 `manager_utils.py`
- 避免重复代码，符合 DRY 原则

### 3. **易于扩展**
- 添加新功能只需修改对应模块
- 不影响其他模块的稳定性

### 4. **清晰的依赖关系**
- 通过明确的导入语句，依赖关系一目了然
- 便于理解模块间的交互

### 5. **向后兼容**
- 保留了 `manager_windows.py` 作为入口模块
- 原有导入方式仍然有效（通过统一导入）

## 📝 注意事项

1. **文件位置**：所有新文件必须与 `LDD.py` 在同一目录下
2. **数据文件**：JSON 数据文件位于 `data/` 子目录中
3. **音频文件**：自定义音频文件放在 `alarm_sounds/` 文件夹中
4. **导入顺序**：确保 `manager_utils.py` 在其他模块之前被正确导入
